### 第一张图：高一下期数学期末复习题一（单选1-8）

**一、选择题（单选）**

**1. 计算复数 `5/(2+i)` = ( )**  
A. 2+i  B. 2-i  C. -2+i  D. -2-i  
**答案：B**  
- **勾画**：分子分母同乘共轭复数 `(2-i)`。  
- **批注**：原式 = `5(2-i)/[(2+i)(2-i)]` = `5(2-i)/5` = `2-i`。  
  错误选项A错在符号；C、D错在实部符号。

**2. 已知不共线向量 `e1,e2`，`a=e1-2e2`，`b=λe1+4e2`，且 `a∥b`，则 λ= ( )**  
A. -2  B. -1  C. 1  D. 2  
**答案：A**  
- **勾画**：利用向量平行（共线）的充要条件：对应坐标成比例。  
- **批注**：设 `a=1·e1+(-2)e2`，`b=λe1+4e2`，由 `a∥b` 得 `1/λ = -2/4`，解得 `λ=-2`。  
  若误用内积为零会错选。

**3. 在△ABC中，AB=15，AC=9，cos∠BAC=3/5，则 BC= ( )**  
A. 9  B. 10  C. 12  D. 15  
**答案：C**  
- **勾画**：已知两边及其夹角，用余弦定理。  
- **批注**：`BC² = AB²+AC²-2·AB·AC·cosA = 225+81-2·15·9·(3/5)=306-162=144`，故 `BC=12`。  
  A(9)是AC长，易误选。

**4. 抽样调查，变量 `xi`（达标为1，不达标为0），则 `(1/100)Σxi` 表示（ ）**  
A. 全校达标人数  B. 样本达标人数  C. 全校达标率  D. 全校达标率的估计值  
**答案：D**  
- **勾画**：理解样本均值与总体参数估计的关系。  
- **批注**：`(1/100)Σxi` 是样本均值，即样本达标率，用于估计全校达标率，不是准确值。  
  C项是参数真值，不选。

**5. △ABC中，D是AC中点，∠CBD=30°，∠ABD=45°，则 AB/BC= ( )**  
A. √2  B. √2/2  C. √3  D. √3/3  
**答案：B**  
- **勾画**：中线、角度关系，可选用正弦定理或面积法。  
- **批注**：由D为中点，△ABD与△CBD面积相等。利用面积公式：  
  `(1/2)AB·BD·sin45° = (1/2)BC·BD·sin30°`，  
  约去BD和1/2得 `AB·(√2/2) = BC·(1/2)`，所以 `AB/BC = √2/2`。  
  A是倒数关系，易错。

**6. 等边△ABC，AC⊥l于A，AC=2，绕l旋转一周所得几何体表面积是（ ）**  
A. 10π  B. 11π  C. 12π  D. 13π  
**答案：C**  
- **勾画**：画图分析旋转体由三部分构成：AC转成圆面，AB转成圆锥侧面，BC转成圆台侧面。  
- **批注**：以A为原点，l为y轴，AC沿x轴，C(2,0)，B(1,√3)。  
  - AC旋转得底面圆面积：π·2²=4π。  
  - AB旋转侧面积：∫₀¹ 2πx·2dx = 2π。  
  - BC旋转侧面积：∫₁² 2πx·2dx = 6π。  
  总面积 = 4π+2π+6π = 12π。  
  只算侧面忘了底面会错选8π等。

**7. 样本空间Ω，n(Ω)=24，n(A)=12，n(B)=8，n(A∪B)=16，则（ ）**  
A. 事件A与B互斥  
B. P(ĀB)=1/2  
C. 事件A与B相互独立  
D. P(ĀB+A B̄)=2/3  
**答案：C**  
- **勾画**：计算 `n(A∩B)=12+8-16=4`。  
- **批注**：A：互斥要求 A∩B 为空，有4个元素，不互斥。  
  B：`n(ĀB)=n(B)-n(A∩B)=4`，P=4/24=1/6，≠1/2。  
  C：`P(A)P(B)=(12/24)×(8/24)=1/6`，而 `P(A∩B)=4/24=1/6`，相等，故独立。√  
  D：`P(ĀB+A B̄)= (4+8)/24=1/2`，≠2/3。

**8. 已知 `|OA|=OA·OB=2`，`OA·OC=6`，则 `|BC|` 的最小值为（ ）**  
A. 1  B. √2  C. 2  D. 4  
**答案：C**  
- **勾画**：用向量坐标化处理。  
- **批注**：设 `OA=(2,0)`。由 `OA·OB=2` 得B的横坐标为1，设 `B(1, y)`；  
  由 `OA·OC=6` 得C横坐标为3，设 `C(3, z)`。  
  则 `|BC|²=(3-1)²+(z-y)²=4+(z-y)² ≥ 4`，当 `z=y` 时取最小值2。  
  误用不等式可能得到其他结果。

**9. 复数 z=3-4i，则（ ）**  
A. z的虚部为-4i  
B. |z|=5  
C. z的共轭为3+4i  
D. z对应点位于第二象限  
**答案：BC**

  B：|z|=√(3²+(-4)²)=5，正确。  
  C：共轭复数实部不变，虚部变号，3+4i，正确。

**10. 徒步统计图题，已知2025高明线人数是2016年的2倍，则（ ）**  
A. 2016年总参与人数20万  
B. 2025顺德线人数超2016南海+顺德总和  
C. 五线增加人数最少的是三水线  
D. 五线增长率最高的是南海线  
**答案：ABD**  
- **批注**：由2025高明3万，得2016高明1.5万，占7.5%，总人数=1.5/7.5%=20万，A对。  
  2016南海线20×30%=6万，顺德线20×25%=5万，和11万，2025顺德12万>11万，B对。  
  增加人数：高明1.5万最少，三水2万，C错。  
  增长率：南海9/6=150%最高，D对。  
  在图上标出对应百分比和数量。

**11. 在△ABC中，AB=3，AC=5，∠BAC=120°，O为各种心，则（ ）**  
A. 垂心：AO·BC=0  
B. 重心：AO·BC=5  
C. 外心：AO·BC=8  
D. 内心：AO·BC=1  
**答案：ACD**  
- **批注**：A：垂心，AO⊥BC，点积为0，正确。  
  B：重心 `AG=(AB+AC)/3`，`AG·BC=(|AC|²-|AB|²)/3=(25-9)/3=16/3≠5`，错。  
  C：建系求外心，或由投影关系解向量系数，得 `AO·BC=8`，对。  
  D：内心 `AI=(5AB+3AC)/15`，计算得 `AI·BC=1`，对。  
  解答时在图上标注向量分解。

**三、填空题**

**12. 已知α,β,γ是三个不同平面，α⊥β，则“γ∥α”是“γ⊥β”的______条件。**  
**答案：充分不必要**  
- **批注**：由面面平行与垂直的性质，若γ∥α且α⊥β，则γ⊥β，充分性成立；  
  若γ⊥β且α⊥β，γ与α可相交，必要性不成立。

**13. 力 `F1=(2,-1)`, `F2=(4,2)`，位移 `s=(2,1)`，合力做功为______。**  
**答案：13**  
- **批注**：合力 `F=F1+F2=(6,1)`，做功 `W=F·s=6×2+1×1=13`。

**14. 正四面体内部有一半径√6的小球，棱长最小值为______；若小球可在内部任意滚动，小球与四面体所有接触点轨迹图形面积为64√3，则棱长为______。**  
**答案：12** ， **20**  
- **详细思路**：  
  正四面体棱长a，内切球半径 `r = √6/12 a`。令 `r≥√6` 得 `a≥12`，故第一空12。  
  小球任意滚动且保持接触，接触点轨迹是各面上缩进的小等边三角形，总面积=`√3 a'²`，其中 `a' = a - 2√6 r`。  
  令 `√3 (a - 2√6·√6)² = 64√3` ⇒ `a - 12 = 8` ⇒ `a=20`。第二空20。

### 第三张图：解答题15、16、17

**15. 直四棱柱，底面ABCD为平行四边形，M为AA₁中点。**  
**(1) 求证：A₁C∥平面BMD。**  
- 连接AC交BD于O，则O为AC中点。  
- 在△A₁AC中，O、M分别是AC、AA₁中点，故OM∥A₁C。  
- OM ⊂ 平面BMD，所以A₁C∥平面BMD。  
**(2) 若BD⊥A₁C，证明底面ABCD为菱形。**  
- 由(1)A₁C∥OM，得BD⊥OM。  
- 直棱柱有AA₁⊥底面，故AA₁⊥BD。  
- 从而BD⊥平面A₁AC，于是BD⊥AC。  
- 平行四边形对角线互相垂直，必为菱形。

**16. 停车收费频率分布直方图。**  
**(1) 估计停车费为24元的频率。**  
- 停车费24元对应时长>6小时，(6,7]、(7,8]频率均为0.05，(8,9]约为0。  
- 频率 = 0.05+0.05 = 0.10。  
**(2) 估计停车时长的第85百分位数。**  
- 累积频率：[0,1]0.25， (1,2]0.45，(2,3]0.60，(3,4]0.70，(4,5]0.80，(5,6]0.85。  
- 第85百分位数落在(5,6]，值为 `5 + (0.85-0.80)/0.05 ×1 = 6` 小时。  
**(3) 800辆车一天停车费收入。**  
- 各区间平均费用：0元(0.25)，4元(0.2)，8元(0.15)，12元(0.1)，16元(0.1)，20元(0.05)，24元(0.1)。  
- 平均停车费 = 0×0.25+4×0.2+8×0.15+12×0.1+16×0.1+20×0.05+24×0.1 = 8.2 元。  
- 总收入 ≈ 800×8.2 = 6560 元。

**17. 三角形ABC，中线AM、BN交于P，AB:AM:AC=6:7:10。**  
**(1) 用AB、AC表示AM、BN。**  
- `AM = (AB+AC)/2`。  
- N为AC中点，`BN = AN - AB = AC/2 - AB`。  
**(2) 求∠BAC。**  
- 设AB=6k，AM=7k，AC=10k。由AM表达式平方：  
  `49k² = 1/4(36k²+100k²+120k²cosA)` ⇒ `cosA=1/2` ⇒ ∠BAC=60°。  
**(3) 若AM·BN=-2，求四边形PMCN的面积。**  
- 计算 `AM·BN`：代入向量，利用AB·AC=30k²，得 `-k²/2 = -2` ⇒ `k=2`。  
- 故AB=12，AC=20，S△ABC=1/2·12·20·sin60°=60√3。  
- 四边形PMCN为两中线围成的两块面积，其和为△ABC面积的1/3，即20√3。

### 第四张图：解答题18、19

**18. 解三角形，a,b,c为角A,B,C的对边。**  
**(1) 由 `asinB = bsin2A` 求A。**  
- 边化角：`sinA sinB = 2 sinB sinA cosA`，消去得 `cosA=1/2` ⇒ A=60°。  
**(2) 若 `b/cosB = √3 c/sinC` 且 `3AC = AB + 2AD`，求 AD/AB。**  
- 由条件化得 `tanB=√3` ⇒ B=60°，结合A=60°得等边三角形。  
- 设边长为1，由向量等式得 `AD = (3AC - AB)/2`。  
- 建系计算得 `|AD|=√7/2`，故 AD/AB = √7/2。  
**(3) 锐角三角形，BC边上高h=2，求面积的取值范围。**  
- 面积 S = 1/2·a·2 = a。  
- 由面积公式与余弦定理得关系 `bc=4a/√3`，`b²+c²=a²+4a/√3`。  
- 锐角条件等价于 `b<2c` 且 `c<2b`，转化得 `2 ≤ (a√3)/4 +1 < 2.5`。  
- 解得 `4/√3 ≤ a < 2√3`，故面积范围 `[4√3/3, 2√3)`。

**19. 梯形翻折问题，AB∥CD，BC⊥AB，BC=CD=2，AB=4，△BCD翻折至△BPD。**  
**(1) 平面ABD⊥平面BPD时：**  
  (i) 证 AD⊥BP：原梯形中可算AD⊥BD，又BP⊥PD。由面面垂直性质，AD⊥平面BPD，故AD⊥BP。  
  (ii) 点B到平面APD的距离：建系，得平面APD方程 `x-z+√2=0`，B(√2,0,0)代入距离公式得距离=2。  
**(2) 直线AP与平面ABD所成角正弦的最大值。**  
- 以BD中点O为原点建系，A(-√2,2√2,0)，P(0,√2 cosθ, √2 sinθ)。  
- 正弦值 = `|√2 sinθ|/√(12-8cosθ)`，令t=cosθ，转化为函数 `g(t)=(1-t²)/[2(3-2t)]` 求最大值。  
- 在 t=(3-√5)/2 时取得，最大正弦值 = `√(3-√5)/2`。

### 第五、六张图：化学原电池方程式书写

**【形成原电池的条件】**  
1. 有两种活动性不同的电极（或导电非金属）  
2. 电极均插入电解质溶液中  
3. 形成闭合回路  
4. 有自发的氧化还原反应  

**【电极式及总反应】**

1. 铜锌原电池（稀硫酸）  
   负极：`Zn - 2e⁻ = Zn²⁺`  
   正极：`2H⁺ + 2e⁻ = H₂↑`  
   总反应：`Zn + 2H⁺ = Zn²⁺ + H₂↑`

2. 铜锌原电池（硫酸铜）  
   负极：`Zn - 2e⁻ = Zn²⁺`  
   正极：`Cu²⁺ + 2e⁻ = Cu`  
   总反应：`Zn + Cu²⁺ = Zn²⁺ + Cu`

3. 锌锰干电池（氯化铵）  
   负极：`Zn - 2e⁻ = Zn²⁺`  
   正极：`2MnO₂ + 2NH₄⁺ + 2e⁻ = 2MnO(OH) + 2NH₃`  
   总反应：已给出

4. 锌锰干电池（氢氧化钾）  
   负极：`Zn - 2e⁻ + 2OH⁻ = Zn(OH)₂`  
   正极：`2MnO₂ + 2H₂O + 2e⁻ = 2MnO(OH) + 2OH⁻`  
   总反应：已给出

5. 铅蓄电池（稀硫酸）  
   负极：`Pb - 2e⁻ + SO₄²⁻ = PbSO₄`  
   正极：`PbO₂ + 2e⁻ + SO₄²⁻ + 4H⁺ = PbSO₄ + 2H₂O`  
   总反应：`Pb + PbO₂ + 2H₂SO₄ = 2PbSO₄ + 2H₂O`

6. 氢氧燃料电池（酸性）  
   负极：`H₂ - 2e⁻ = 2H⁺`  
   正极：`O₂ + 4e⁻ + 4H⁺ = 2H₂O`  
   总反应：`2H₂ + O₂ = 2H₂O`

7. 氢氧燃料电池（碱性）  
   负极：`H₂ - 2e⁻ + 2OH⁻ = 2H₂O`  
   正极：`O₂ + 4e⁻ + 2H₂O = 4OH⁻`  
   总反应：`2H₂ + O₂ = 2H₂O`

8. 甲烷燃料电池（酸性）  
   负极：`CH₄ - 8e⁻ + 2H₂O = CO₂ + 8H⁺`  
   正极：`2O₂ + 8e⁻ + 8H⁺ = 4H₂O`  
   总反应：`CH₄ + 2O₂ = CO₂ + 2H₂O`

9. 甲烷燃料电池（碱性）  
   负极：`CH₄ - 8e⁻ + 10OH⁻ = CO₃²⁻ + 7H₂O`  
   正极：`2O₂ + 8e⁻ + 4H₂O = 8OH⁻`  
   总反应：`CH₄ + 2O₂ + 2OH⁻ = CO₃²⁻ + 3H₂O`

10. 镁铝电池（稀硫酸）  
    负极：`Mg - 2e⁻ = Mg²⁺`  
    正极：`2H⁺ + 2e⁻ = H₂↑`  
    总反应：`Mg + 2H⁺ = Mg²⁺ + H₂↑`

11. 镁铝电池（氢氧化钠溶液）  
    负极：`Al - 3e⁻ + 4OH⁻ = AlO₂⁻ + 2H₂O`  
    正极：`2H₂O + 2e⁻ = H₂↑ + 2OH⁻`  
    总反应：`2Al + 2OH⁻ + 2H₂O = 2AlO₂⁻ + 3H₂↑`
